# Oropeza-Post1-U7
Gestión de Productos — Spring Boot + Thymeleaf
Aplicación web CRUD para administrar un catálogo de productos, desarrollada con Spring Boot 3.2, Thymeleaf y persistencia en memoria.
Implementa el patrón MVC y el flujo Post/Redirect/Get (PRG) en todas las operaciones de escritura.

Cómo ejecutar
1. Clonar el repositorio Oropeza-post1-u7
2. Ejecutar con el wrapper de Maven
./mvnw spring-boot:run
O si Maven está instalado globalmente:
mvn spring-boot:run
3. Abrir en el navegador "http://localhost:8080/productos"
